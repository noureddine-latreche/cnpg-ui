About Settings Storage

Settings are stored in a SQLite database at the path configured by the DB_PATH environment variable (default: /data/cnpg-ui.db). In production (Helm), this path is backed by a Persistent Volume Claim so settings survive pod restarts.

AWS access keys for S3 scanning are read from the AWS_ACCESS_KEY_ID and AWS_SECRET_ACCESS_KEY environment variables (or instance profile if running on EC2/EKS). They are not stored in the database.
